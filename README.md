# Layout
- Background black
- spacegray theme
- NerdTree colorized

# Supported lang
- .php
- .js
- .vue
- .ts

# Control
- Ctrl + B to open NerdTree
- Ctrl + I to show indent lines
- Ctrl + P to open file fuzzy finder
- Ctrl + H to move left window split
- Ctrl + L to move right window split
- Automatically add close bracket
